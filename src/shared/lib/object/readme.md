### object

Набор функций-хелперов для работы с объектами:

1. `getObjectKeys` - Возвращает типизированный массив ключей объекта
2. `getObjectEntries` - Возвращает типизированный массив записей [ключ, значение] для объекта
3. `pick` - Создает новый объект, выбирая только указанные ключи из исходного объекта
4. `omit` - Создает новый объект, исключая указанные ключи из исходного объекта
5. `toFormData` - Конвертирует объект в FormData
