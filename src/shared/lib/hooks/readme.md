## hooks

Набор react-хуков

1. useEvent - Подробнее смотри тут: https://github.com/reactjs/rfcs/blob/useevent/text/0000-useevent.md
2. useDebouncedValue - Возвращает передаваемое значение с debounce-эффектом
3. useMount - Lifecycle-hook, который вызывает коллбек при монтировании компонента
4. useUnmount - Lifecycle-hook, который вызывает коллбек при размонтировании компонента
